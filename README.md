#RestaurantIQ 
###RestaurantIQ aggregates reviews from all the popular platforms (Foursquare, Yelp, etc.) and reservation links in one easy to use app.

###Built Using: 
  * React
  * Redux
  * Express
  * Material Design
  * (Factual API)[http://developer.factual.com/]
